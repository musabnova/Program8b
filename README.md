# Program 8b - Cookie Visit Counter Servlet

## Objective
Build a servlet program to create a cookie using user input and track the number of visits. Display greeting message, visit count, list of cookies and demonstrate cookie expiry.

## Files
- CookieServlet.java
- index.html

## Source Code
- Servlet: https://github.com/musabnova/Program8b/blob/master/src/main/java/com/cookie/CookieServlet.java
- HTML: https://github.com/musabnova/Program8b/blob/master/src/main/webapp/index.html

## Features
- Takes user name from text box
- Creates cookie
- Tracks number of visits
- Displays greeting message
- Shows all active cookies
- Demonstrates cookie expiry

## How to Run
1. Run project on Apache Tomcat
2. Open: http://localhost:8080/Program8b/
3. Enter your name and submit
4. Refresh page to increase visit count
